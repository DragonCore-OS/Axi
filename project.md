## T0 - 时间盒规则 (Time-Box Rule)

### 规则定义

**所有实验不得超过现实时间 1 小时。**

### 申请延长

1 小时后如果有**正向反馈**，可以申请再增加 **1-2 小时**。

### 申请流程

```
实验运行 1 小时
    ↓
[检查点] 评估正向反馈
    ↓
├── 无正向反馈 → 实验终止
│
└── 有正向反馈 → 提交延长申请
            ↓
        说明：
        - 已获得的具体成果
        - 延长后的预期目标
        - 申请延长时间（1小时或2小时）
            ↓
        [批准] → 继续实验
        [拒绝] → 实验终止，记录结果
```

### 正向反馈标准

| 类型 | 示例 |
|------|------|
| 技术验证 | 核心算法通过测试 |
| 数据确认 | 实验假设得到数据支持 |
| 问题定位 | 阻塞问题已识别并有解决方案 |
| 性能突破 | 关键指标达到预期 |

### 强制执行

- ⏰ **1 小时硬截止**：无论状态如何，必须停止并汇报
- 📝 **强制记录**：终止时必须记录当前状态、已得成果、未完成项
- 🚫 **无例外**：未经申请的延长视为违规
